Project Overview

NomNoms is a restaurant discovery app designed to create a more transparent and personalized experience for both foodies and restaurant owners.

The platform supports separate login and signup flows for users and business owners, securely storing account information so users can return with their profiles and activity preserved. Restaurant owners can create and manage restaurant profiles, while users can explore restaurants and reviews, leave comments, and submit ratings.

NomNoms features live restaurant search that updates results as users type, making it easy to quickly find registered restaurants by name. Users can also filter restaurants using descriptive tags such as cuisine or restaurant type, with each restaurant card displaying its associated tags for easy comparison.

Users can rate restaurants using a 5-star system and leave comments, with average ratings and feedback updating in real time for both users and owners to view. To ensure privacy and security, the app uses token-based authentication that identifies users by ID rather than username, with tokens automatically expiring after one hour.
