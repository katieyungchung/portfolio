Journaling often feels effortful, repetitive, or unrewarding, and existing tools lack engaging structures that encourage reflection, growth, and follow-through.

Motivation

Studies have shown that regularly reflecting on both positive experiences and challenges can improve emotional well-being. Rose Garden makes journaling easier and more engaging by using the roseâ€“budâ€“thorn reflection framework, where roses represent positive moments, buds represent things to look forward to, and thorns represent challenges or difficulties. Through these prompts, users plant their reflections in a digital garden, rewarding consistency and encouraging long-term reflection through visible growth and progress.

Key Features

Onboarding

When first opening the app, users are guided through tutorial pages, prompted to create an account, and asked to select initial journaling tags.


Journaling User Flow

Each day, the user can select which of the rose, bud, thorn prompts they would like to journal for. Users fill out each entry they select, and add a title and custom tags. The user can also customize their journaling flower to plant in their digital garden.


FreeDraw

This feature was built with React SVG + Gesure Handler, and allows for freehand drawing text input and image upload. The user can modify stroke width, select colors with the color wheel picker, undo, and pan/pinch/rotate and delete pictures.


Calendar/Garden View

In the calendar view, users can view journal entries by month. In the Garden View, users can grow flowers by filling out journal entries. The garden was implemented using Skia Canvas and React Native Reanimated. The profile page contains user info and unlocked flowers.
